# Mini Project Frontend

This is the frontend part of the mini project, built using Angular. The application includes a login page and a dashboard that displays user-specific data after authentication.

## Project Structure

- `src/app/login`: Contains the login component for user authentication.
- `src/app/dashboard`: Contains the dashboard component that displays data after login.
- `src/app/app.module.ts`: The main module that imports necessary Angular modules and declares components.
- `main.ts`: The entry point of the Angular application.

## Setup Instructions

1. **Install Dependencies**: Run `npm install` in the `frontend` directory to install all required packages.
2. **Run the Application**: Use `ng serve` to start the development server. The application will be available at `http://localhost:4200`.

## Features

- User authentication through a login form.
- A dashboard that displays data after successful login.

## Notes

Ensure that the backend API is running and accessible for the frontend to function correctly.
